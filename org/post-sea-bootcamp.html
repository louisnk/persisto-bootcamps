Post SEA Bootcamp

(Secondary Entrance Assessment)

8 weeks of Learning

Coding fundamentals

Using block programming and a pre-designed course, students moved from zero knowledge  to working with the basics. At the end of 4 weeks the entire class was using control flow, variables, loops, and simple functions.

Intro to Python

After completing the introductory course, we moved on to translating those concepts into actual code in the Python language. We did this using Raspberry Pi computers, a robot car kit, and Minecraft Pi Edition.

Python in Robotics

Some students focused on learning to write code to create things in Minecraft, while others tried to program a robotic car to drive itself safely through an obstacle course. In the end, all covered similar concepts.

Learning Outcomes

Cross Discipline

  • Explain and apply variables
  • Create and solve logic problems
  • Solve complex problems using simple logical steps
  • Recognize small, repeatable subsets within larger processes
  • Explain the concept of a function with variable input
  • Describe a 3-dimensional grid, using X, Y, and Z coordinates
  • Demonstrate ability to learn independently from online sources

Computer Science

  • Explain and apply various data types
  • Use control flow statements to build simple algorithms
  • Understand and utilize various types of loops
  • Illustrate the use of loops to repeat sub-processes
  • Explain and use functions to encapsulate repeated tasks
  • Demonstrate the use of loops, control flow, and functions to create more complex algorithms
  • Practice debugging techniques to investigate and solve errors
  • Understand and differentiate basic data structures
